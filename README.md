## Installation

-   `git clone https://github.com/fiqih16/kledo_backend_laravel.git`
-   `cd kledo_backend_laravel`
-   `cp .env.example .env`
-   `composer updat`
-   Edit `.env`
    -   Set your database connection details
-   `php artisan migrate`
-   `php artisan db:seed`
-   `php artisan serve`
